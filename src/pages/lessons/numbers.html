Les Nombres et Calculs en JavaScript

En JavaScript, les nombres sont utilisés pour effectuer des calculs simples ou complexes. Voyons ensemble les bases : opérations, modulo, incrémentation et opérateurs d’assignement composés.

Déclarer des nombres

En JavaScript, les nombres peuvent être entiers ou à virgule flottante :

let entier = 42;
let flottant = 3.14;

Opérations de base

Les 4 opérations classiques :

let a = 10;
let b = 3;

console.log(a + b); // 13
console.log(a - b); // 7
console.log(a * b); // 30
console.log(a / b); // 3.333...

Le Modulo (%)

Le modulo donne le reste d’une division :

console.log(10 % 3); // 1
console.log(15 % 4); // 3

Incrémenter / Décrémenter

On peut ajouter ou retirer 1 à une variable :

let x = 5;
x++; // équivaut à x = x + 1
console.log(x); // 6

x--; // équivaut à x = x - 1
console.log(x); // 5

Opérateurs d’assignement composés

Ils permettent de raccourcir l’écriture :

let y = 10;
y += 5; // équivaut à y = y + 5 → 15
y -= 2; // équivaut à y = y - 2 → 13
y *= 2; // équivaut à y = y * 2 → 26
y /= 2; // équivaut à y = y / 2 → 13

⚠️ Attention aux décimales

JavaScript utilise le type Number basé sur les flottants.

console.log(0.1 + 0.2); 
// 0.30000000000000004

Solution : utiliser toFixed() ou des librairies.
